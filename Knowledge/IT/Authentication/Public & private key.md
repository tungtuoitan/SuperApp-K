# private key và public key dùng để làm gì? [id:48 order:2]
- mã hóa dữ liệu để bảo mật
- kí token

# ví dụ pattern ngoài đời thật tương tự private key, public key ? [id:49 order:3]
Private key = con dấu thật của công ty
Public key = mẫu dấu công khai cho mọi người xem

# cách dùng private key, public key để mã hóa? [id:50 order:4]
dùng public để encrypt data, rồi B dùng private key để decrypt data
