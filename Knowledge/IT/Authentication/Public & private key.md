# private key và public key dùng để làm gì? [id:2302 order:1]
- mã hóa dữ liệu để bảo mật
- kí token

# ví dụ pattern ngoài đời thật tương tự private key, public key ? [id:2303 order:2]
Private key = con dấu thật của công ty
Public key = mẫu dấu công khai cho mọi người xem

# cách dùng private key, public key để mã hóa? [id:2304 order:3]
dùng public để encrypt data, rồi B dùng private key để decrypt data
