# .NET là gì?
là nền tảng phát triển phần mềm của Microsoft, gồm runtime (CLR) và thư viện chuẩn (BCL), dùng để build app console, web, desktop, mobile.

# CLR là gì?
là runtime của .NET
, chịu trách nhiệm biên dịch IL sang machine code (JIT), quản lý bộ nhớ (GC), và xử lý exception.

là Garbage Collector — tự động giải phóng memory của object không còn được tham chiếu, không cần dev tự `free()`.
