# Azure Blob Storage là gì? [id:2309 order:1]
Dịch vụ lưu trữ file của Microsoft Azure dùng để lưu file nhị phân (ảnh, PDF, video...) trên cloud.

# File trong Azure Blob có public mặc định không? [id:2310 order:2]
Không.

# Azure Blob có mấy chế độ truy cập? [id:2311 order:3]
Hai chế độ: Public và Private.

# Public và Private khác nhau thế nào? [id:2312 order:4]
Public: ai cũng truy cập được. Private: chỉ owner mới truy cập được.

# Azure Blob có CDN sẵn không? [id:2313 order:5]
Có.

# Bật CDN thì URL thay đổi như thế nào? [id:2314 order:6]
```
Trước: https://mystorage.blob.core.windows.net/files/img.jpg
Sau:   https://mystorage.azureedge.net/files/img.jpg
```
