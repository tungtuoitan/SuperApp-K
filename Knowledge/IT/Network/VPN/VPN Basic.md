# VPN là gì? [id:2634 order:1]
Virtual Private Network — công nghệ tạo kết nối riêng tư, mã hóa đi xuyên qua internet công cộng.

# VPN giải quyết những vấn đề gì? [id:2635 order:2]
Bốn vấn đề: ISP theo dõi traffic, geo-block, rủi ro wi-fi công cộng, và remote access vào mạng nội bộ.

# VPN ẩn IP thật bằng cách nào? [id:2636 order:3]
VPN server thay IP nguồn của client bằng IP của chính nó trước khi forward request đến website đích.

# Bản chất của VPN là gì? [id:2637 order:4]
Mã hóa (bảo vệ nội dung) + encapsulation/tunnel (ẩn metadata, định tuyến lại) + authentication (xác minh ai được kết nối).
