# VPN là gì? [id:453 order:1]
Virtual Private Network — công nghệ tạo kết nối riêng tư, mã hóa đi xuyên qua internet công cộng.

# VPN giải quyết những vấn đề gì? [id:454 order:2]
Bốn vấn đề: ISP theo dõi traffic, geo-block, rủi ro wi-fi công cộng, và remote access vào mạng nội bộ.

# VPN ẩn IP thật bằng cách nào? [id:455 order:3]
gói metadata lại + dùng ip của vpn server khi giao tiếp với website

# Bản chất của VPN là gì? [id:456 order:4]
Mã hóa + encapsulation + authentication
