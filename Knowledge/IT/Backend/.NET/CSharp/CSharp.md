# C# là gì?
là ngôn ngữ lập trình
# C# có phải ngôn ngữ OOP không?
có
# đặc điểm OOP trong C#?
- Encapsulation: che giấu state bằng access modifier
- Inheritance: class con kế thừa class cha (single inheritance)
- Polymorphism: virtual/override để override hành vi
- Abstraction: dùng interface và abstract class
# C# chạy trên nền tảng nào?
chạy trên .NET — code C# được compile thành IL, CLR JIT compile sang machine code lúc runtime.
# Roslyn là gì?
là compiler chính thức của C# và VB trong .NET — biên dịch C# → IL, output `.dll`/`.exe`.
# ý nghĩa tên Roslyn?
Roslyn là tên 1 thành phố ở Washington (Mỹ) — Microsoft đặt tên project theo thói quen lấy tên địa danh.
