# .NET là gì? [id:2690 order:1]
là nền tảng phát triển phần mềm của Microsoft, gồm runtime (CLR) và thư viện chuẩn (BCL), dùng để build app console, web, desktop, mobile.

# CLR là gì? [id:2691 order:2]
là runtime của .NET
, chịu trách nhiệm biên dịch IL sang machine code (JIT), quản lý bộ nhớ (GC), và xử lý exception.

# IL là gì? [id:2693 order:4]
là Intermediate Language — bytecode mà compiler C# tạo ra. CLR sẽ JIT compile IL thành machine code lúc runtime.

# JIT là gì? [id:2696 order:7]
là quá trình CLR biên dịch IL sang machine code

# GC trong .NET là gì? [id:2697 order:8]
là Garbage Collector — tự động giải phóng memory của object không còn được tham chiếu, không cần dev tự `free()`.

# namespace trong .NET là gì? [id:2698 order:9]
là 1 nhóm các class/interface

# assembly là gì?
là đơn vị deploy của .NET — thường là file `.dll` hoặc `.exe`, chứa IL và metadata của 1 project hoặc library.

# đơn vị deploy là gì?
là phần nhỏ nhất mà bạn copy/ship lên server để app chạy. Trong .NET là file `.dll`/`.exe`.

# đơn vị deploy hầu hết là file phải không?
Đúng, nhưng cũng có thể là folder, container image (Docker), hoặc package (`.nupkg`, `.zip`) tùy cách ship.

# ship là gì?
là đóng gói và đưa code/sản phẩm lên môi trường thật để user dùng — có thể là deploy server, publish package, push image lên registry.
