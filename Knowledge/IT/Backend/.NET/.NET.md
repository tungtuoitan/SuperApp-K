# delegate giờ ít phổ biến à? vì sao? [id:3257 order:6]
đúng, vì giờ hầu hết dùng Action
Ít dùng trực tiếp. Thay vào đó dùng `Action`, `Func<T>` (built-in delegate) hoặc lambda trực tiếp. Từ khoá `delegate` để khai báo type mới thì hiếm — chỉ khi cần tên rõ ràng hơn `Action`/`Func`.

# Task.WhenAny hoạt động thế nào?
trả về ngay khi có **một** task hoàn thành đầu tiên (không cancel các task còn lại).

# Retry external service là pattern à?
Đúng — "Retry with exponential backoff" là pattern chuẩn khi gọi external service (DB, API, queue).
# Retry with exponential backoff pattern giải quyết vấn đề gì?
external service có thể tạm thời unavailable (transient failure). 
Retry với delay tăng dần (200ms → 400ms → 800ms) 
->> tránh hammer server khi nó đang phục hồi.
# So sánh ReadAllTextAsync và StreamReader?
`File.ReadAllTextAsync` load **toàn bộ** file vào RAM một lúc 
`StreamReader` đọc từng chunk
