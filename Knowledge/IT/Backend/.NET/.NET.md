# delegate giờ ít phổ biến à? vì sao? [id:3257 order:6]
đúng, vì giờ hầu hết dùng Action

# composition over inheritance nghĩa là gì?
# Vì sao "Composition over Inheritance"
vì kế thừa tạo coupling cứng: đổi cha là con vỡ, và chỉ kế thừa được 1 cha. Composition ráp object theo interface → linh hoạt, test dễ, đổi behavior runtime được.
